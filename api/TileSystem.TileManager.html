Class TileManager

Used to manage and update Unity tile maps and store tile data

Inheritance
System.Object
TileManager
Namespace: TileSystem
Assembly: cs.temp.dll.dll
Syntax
public class TileManager : MonoBehaviour
Remarks

Can store multiple layers of tilemaps, these are gotten from the children of the grid. The sorting order of the tile renders must be sequential (ie. 1,2,3 is ok but 1,3,4 is not). Tiles are accessed using a Vector3Int, the X and Y cords correspond to the X and Y position of the tile and Z is used for the layer.
