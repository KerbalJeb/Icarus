Class Damage

A basic class used to describe and apply damage to tilemaps

Constructors

Damage(Vector2, Vector2, Single)

Basic constructor for damage, used to apply damage in a line. Will damage all tiles hit until the base damage has been depleted (ie. if basedmg=100 and it hits a tile with 50hp and then a tile with 100hp it will destroy the first tile and do 50 dmg to the second).
